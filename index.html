Summary: Latitude versus X

Does weather change as one gets closer to the equator? To figure out if this is true we pulled data from the OpenWeatherMapAPI to assemple a dataset on >500 cities.

We used MatPlotlib on the dataset to plot weather conditions versus latitude.The factors used included: temperature, cloudiness, windspeed, and humidity. This site provides source data, visualizations, explanations, and correlations.
